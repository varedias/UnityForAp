## 方案2：切换回旧输入系统（可选）

如果您不想使用新的 Input System，可以切换回旧版：

1. 打开 `Edit > Project Settings > Player`
2. 找到 `Other Settings > Active Input Handling`
3. 改为 `Input Manager (Old)` 或 `Both`
4. 重启 Unity 编辑器

---

## 方案3：完全使用新的 Input System（高级）

如果想完全使用新系统：

1. 确保已安装 Input System 包
   `Window > Package Manager > Input System > Install`

2. 使用我创建的 InputManager，它已经包含新系统支持

3. 设置为 `Input System Package (New)`

## InputManager 支持的功能

- `InputManager.GetSpaceKeyDown()` - 空格键
- `InputManager.GetKeyDown(KeyCode)` - 任意键
- `InputManager.GetMouseButtonDown(int)` - 鼠标点击
- `InputManager.GetMousePosition()` - 鼠标位置
- `InputManager.GetTouchDown()` - 触摸输入

可在任何脚本中使用，自动兼容新旧系统！
